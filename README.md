# Medical Center Management System

## Project Description
This project is a Software Engineering course project for a Medical Center Management System.  
The system helps a medical center manage patients, doctors, appointments, consultations, prescriptions, payments, and reports.

## Prototype Type
This prototype is implemented as a C++ console application.  
It demonstrates the main features of the Medical Center Management System.

## Tools and Technologies
- C++17
- Visual Studio Code or Visual Studio
- g++ compiler

## Main Features
- User login
- Patient registration
- Patient search
- Appointment booking
- Double-booking prevention
- Patient check-in
- Doctor consultation recording
- Prescription recording
- Payment recording
- Admin report summary

## How to Run the Project

### Using g++
Open the project folder in the terminal and run:

```bash
g++ -std=c++17 main.cpp -o MedicalCenterMVP

```

Then run the program:

```bash
./MedicalCenterMVP
```

On Windows, run:

```bash
MedicalCenterMVP.exe
```

## How to Use
1. Run the program.
2. Log in using one of the demo accounts.
3. Choose the required feature from the menu.
4. Test patient registration, patient search, appointment booking, double-booking prevention, check-in, consultation recording, payment recording, and admin reports.

## Notes
The prototype uses in-memory data for demonstration purposes.  
It does not require an external database or web server.
